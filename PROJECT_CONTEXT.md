## 🎨 最近完成的工作

### 2026-03-12：优化微调面板和整体缩放功能

#### 1. 优化方向键布局

**问题**：
微调面板的方向键是横向排列，不符合用户习惯。

**解决方案**：
改成十字形键盘布局：
```
    ↑
  ← ↓ →
  (x,y)
```

**优势**：
- 更直观，符合键盘方向键认知
- 更紧凑，垂直占用空间更少
- 按钮尺寸从 `32x24` 增大到 `40x32`，点击区域更大

#### 2. 实现整体缩放功能

**问题**：
锁定图层后无法缩放，但用户希望在两个图层都锁定后能一起缩放。

**解决方案**：
- 添加 `areAllLayersLocked` 计算属性，检测是否所有可见图层都已锁定
- 当所有图层都锁定时，启用整体缩放手势
- 添加缩放比例实时显示提示

**使用场景**：
1. 用户精确对齐两个图层后
2. 锁定两个图层，防止误操作
3. 双指缩放，整体放大或缩小查看细节

**实现细节**：
- 整体缩放范围：0.5x - 3.0x
- 缩放时显示实时百分比提示
- 重置图层时同步重置整体缩放

   - 左右标签显示图层名称

3. 在 toolbar 添加模式切换 Segmented Control

4. 分屏模式下禁用微调面板（不需要微调）
